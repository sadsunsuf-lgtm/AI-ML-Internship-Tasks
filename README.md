## 📂 Task 4: AI Health Assistant (Groq & Next.js)
**🎯 Objective:** To build a high-speed, interactive AI chatbot for general health inquiries.
**🛠️ Tools & Libraries:** Next.js 15, React, Groq API (Llama 3.3 70B), Tailwind CSS.
**📈 Key Features:** * **Speed:** Leverages Groq's LPUs for near-instant responses.
* **Safety:** Implemented system prompts to include medical disclaimers.
* **Modern UI:** Responsive design with professional gradients and loading states.
